Proton Synchrotron Optics Repository

Home
This website contains the official optics models for the CERN Proton Synchrotron. For each scenario listed below (and the various configurations belonging to each scenario), MAD-X input scripts, twiss tables and optics plots are available and can be browsed.

Operational optics scenarios

  • bare machine - Combined function magnets only.
  • AD - Proton beams for the Antiproton Decelerator.
  • EAST - Proton beams for the EAST area.
  • LHC ION - Lead ion beams for the LHC physics programme.
  • LHC PROTON - Proton beams produced for the LHC physics programme.
  • SFTPRO - Proton beams produced for the SPS fixed target physics programme.
  • TOF - Proton beams for the n_TOF fixed target physics programme.

Data structure

The data for the different operational scenarios quoted above are organized in the following way:
  • Optics scenario: each scenario corresponds to a certain operational cycle (i.e. beam and/or user).
  • Configuration: the state of the machine for a specific instant along the cycle (e.g. injection, flat bottom, ...).
For each configuration the values of several parameters (energy, transverse tunes, optical functions at the various BI monitors) as well as Twiss tables and plots of the optics functions are provided.
